Frontend Developer (Flutter/Dart + React)
Job description

We are seeking a frontend developer skilled in building high-quality mobile apps with Flutter/Dart and responsive web applications with React. You will own end-to-end UI development across platforms, collaborate closely with design and backend teams, and deliver performant, accessible, and maintainable user experiences. The ideal candidate is product-minded, tests thoughtfully, and ships iteratively.

Responsibilities
  • Design, build, and maintain Flutter/Dart mobile applications (iOS/Android) with scalable architecture and modular components.
  • Develop responsive, accessible web UIs using React, modern hooks, and component-driven development.
  • Translate Figma/Design specs into clean UI with consistent theming, animations, and interactions.
  • Implement client-side state management, caching, and data synchronization patterns.
  • Integrate REST/GraphQL APIs, handle authentication flows, and manage secure client storage.
  • Implement multi-language support (EN/DE)
  • Ensure stable performance, accessibility (WCAG), and reliability across devices/browsers.
  • Write automated tests (unit, widget/component, integration, e2e) and participate in code reviews.
  • Collaborate with product/design on requirements, feasibility, and iteration; provide technical feedback and estimates.
  • Maintain CI/CD pipelines for mobile and web delivery; support store submissions and web deployments.
  • Monitor production health (crash/error analytics, performance metrics) and drive continuous improvements.
  • Document architecture, components, and development practices; contribute to shared UI libraries/design systems.
Requirements
  • Professional experience shipping Flutter/Dart mobile apps and React web applications.
  • Strong proficiency in TypeScript/JavaScript and Dart; solid understanding of modern React (hooks, context), Flutter widgets, layout, and navigation.
  • Familiarity with state management:
    • Flutter: Provider, Riverpod, Bloc/Cubit, or similar
    • React: Zustand, Redux, or React Query/TanStack Query as applicable
  • Experience integrating APIs (REST/GraphQL), handling offline scenarios, optimistic updates, and error states.
  • Solid grasp of responsive design, accessibility standards, and cross-browser/device compatibility.
  • Competence with testing:
    • Flutter: unit/widget tests (flutter_test), integration tests
    • React: Jest, Testing Library, Playwright/Cypress
  • Experience with build and release tooling: Flutter build pipelines, Fastlane, Firebase/App Distribution/TestFlight/Play Console; Web bundlers (Vite/Next), CI/CD (e.g., GitHub Actions).
  • Version control (Git), code review workflows, and collaborative development practices.
Qualifications
  • Bachelor's degree in Computer Science, Engineering, or equivalent practical experience.
  • Demonstrated portfolio: published apps (App Store/Play Store) and/or production React web projects.
  • Experience with one or more of:
    • React frameworks: Next.js, routing, SSR/SSG, data fetching strategies
    • Design systems and component libraries (e.g., shadcn-ui/Tailwind on web; Flutter theming)
    • State libraries mentioned above; TanStack Query/React Query for data fetching
    • Analytics, crash reporting, and remote config (e.g., Firebase, Sentry)
    • Authentication and security best practices (OAuth, JWT, secure storage, OWASP)
    • Backend integration familiarity (Node/Hono/Supabase/Firebase) is a plus
  • Nice-to-have: experience with animations (Flutter Implicit/Explicit, Rive/Lottie; React Framer Motion), internationalization, and offline-first patterns.
